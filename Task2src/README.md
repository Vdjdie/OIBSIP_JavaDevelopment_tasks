# Number Guessing Game (Java)

## Description
This is a simple Java console-based Number Guessing Game.
The program generates a random number between 1 and 100.
The user keeps guessing until the correct number is found.

## Technologies Used
- Java
- Scanner
- Random

## How to Run
1. Compile the program:
   javac NumberGuessing.java
2. Run the program:
   java NumberGuessing
