# Online Reservation System

## Description
This is a simple **Online Reservation System** created using **HTML, CSS, and JavaScript**.
The project allows users to log in, reserve a ticket, and cancel a ticket through a
user-friendly web interface.

## Features
- Login system with username and password
- Ticket reservation form
- Ticket cancellation using PNR number
- Input validation
- Simple and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Login using the provided credentials
4. Reserve or cancel tickets using the menu

- Ticket reservation confirmation alert
